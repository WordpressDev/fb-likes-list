Displays posts based on Facebook 'like' count

== Description ==

Retrieves FB like count and stores as postmeta to display a 'popular' post widget

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `fb-likes-list` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Drag the widget to the sidebar area of choice

== Frequently Asked Questions ==


= I have posts with a lot of likes that aren't showing =

The post doesn't get a count until it's been loaded on the front-end of the site.
